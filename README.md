### Project overview
Naluri Space Project web app that displays
- The accurate value of Pi
- The circumference of Sun
### Built with
- [Laravel](https://laravel.com)

### Running Locally
Clone the project and make it your working directory:
```sh
git clone https://github.com/edrryy/naluri-coding-challenge.git
cd naluri-coding-challenge
```
Install Dependecies:
```sh
- create .env
- copy .env.example into .env
- composer install
- npm install
```
Run Laravel Project:
```sh
- npm run dev
- php artisan serve
- open http://localhost:8000/
```
